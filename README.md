- Database: MySQL

# 🚀 Installation

```bash
git clone https://github.com/AhmedFarag555/Property-Hub-AI.git
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Swagger:
`http://localhost:8000/docs`
